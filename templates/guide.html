Stock Trading System - Quick Guide

This is a simple guide page added to provide basic usage instructions.

IIFL Bridge Connection

The application uses the IIFL bridge at bridge.iiflcapital.com. The bridge port is configurable via the environment variable IIFL_BRIDGE_PORT (defaults to 8883).

Common Endpoints

  • / - Dashboard
  • /signals - Signals
  • /portfolio - Portfolio
  • /settings - Settings
  • /guide - This guide page
